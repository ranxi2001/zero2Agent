# learn-frameworks

这一部分做横向视角，不追求罗列，而是建立选型标准。

不是所有项目都要上多 Agent，不是所有 Agent 都需要 LangGraph，也不是所有场景都适合复杂框架。理解框架的取舍比背 API 更重要。

## 会比较什么

- 抽象层级
- 状态管理方式
- 工具接入能力
- 调试与可观测性
- 适合做 Demo 还是适合做系统

## 可纳入的对象

- LangChain / LangGraph
- OpenAI Agents SDK
- MCP 生态
- CrewAI / AutoGen / Agno
