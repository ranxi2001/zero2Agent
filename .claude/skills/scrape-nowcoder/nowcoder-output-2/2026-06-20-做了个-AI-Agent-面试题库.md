做了个 AI Agent 面试题库

最近在准备 AI Agent 方向的面试，发现市面上的八股文资料太散了，于是边学边做了一个小网站。

在线访问地址：http://devinterview.cn

先叠甲 🙏：

纯 vibe coding 做的，前端 React + Ant Design，后端 FastAPI + SQLite，基本靠 Cursor / Claude Code 写出来的。
题目来源于网络，主要是牛客网的公开面经和社区讨论，并非原创，
答案都是 AI 生成的，有些可能不准确甚至有错误，仅供参考。

亮点：

搜索： 用了 SQLite FTS5 全文检索，比模糊 LIKE 好一点，搜 "function calling" 能多命中几道相关题
知识库引用： 每道题都关联了agent学习绕不开的几个著名的开源项目，可以直接跳转看原文，评分相关度做的比较简单粗暴（关键词匹配）
完全免费无广告： 纯粹是自己学习用的，没有任何商业化的打算

GitHub 仓库：https://github.com/lxxxxx-oss/interview-master
觉得还可以的话点个 ⭐ Star 支持一下，觉得有问题就提提 Issue，感谢各位大佬！
