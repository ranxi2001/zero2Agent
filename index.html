这个项目适合谁

你已经会用 AI，但不会开发 Agent

学习路线

不是碎片化材料堆砌，而是从 Agent 基础认知开始，逐步进入框架、Code Agent 和真实业务场景。

01

learn-agent-basic

先讲清楚 Agent 是什么、为什么不是普通工作流，以及工具、记忆、规划、RAG 分别解决什么问题。
02

learn-langgraph

从状态图和可恢复执行切入，把真正可维护的 Agent 系统搭起来，而不是停留在线性链式调用。
03

learn-claude-code

聚焦 Code Agent 设计，拆解上下文管理、工具系统、命令执行、补丁写入和任务闭环。
04

learn-frameworks

横向理解不同框架和生态，不追求全部都学，而是建立选型视角和取舍能力。
